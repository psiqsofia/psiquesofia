Bank Marketing Prediction

A machine learning project to predict whether a customer will accept a bank's marketing campaign offer, using real historical data.

Python Pandas Scikit-learn Matplotlib Jupyter
View more

Goal: Build a classification model to predict the likelihood of a client accepting a marketing campaign call.

Preprocessing: Handled categorical variables, normalized data, removed data leakage by excluding 'duration'.

Model Selection: Compared Random Forest classifiers with and without the variable 'duration'.

Evaluation: The model without 'duration' is more realistic and ethical, with balanced F1-score for the minority class.

Insight: Accuracy dropped slightly, but practical applicability improved by focusing on pre-call variables only.

Mystical Experience Prediction at Raves

Neural network model to predict mystical experiences during rave events based on psychological and contextual factors using synthetic data.

Python Keras Neural Network EDA Visualization
View more

Goal: Predict whether a person experiences a mystical state at a rave using emotional, spiritual and contextual inputs.

Model: Feedforward Neural Network trained for 50 epochs with 68% test accuracy.

Evaluation: Confusion matrix shows good identification of 'No' cases but struggles with 'Yes', due to class imbalance.

Insight: Euphoric state, connection with others, and high spirituality are strong predictors.
